# Telegram бот для розыгрыша

Бот проверяет вступление в чат `t.me/torgovlya_kfu` и принимает скрин репоста, чтобы повысить шанс в розыгрыше.

## Быстрый старт
1) Установи зависимости:
```
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
2) Создай файл `.env` и пропиши токен:
```
BOT_TOKEN=YOUR_TELEGRAM_TOKEN
```
3) Запусти бота:
```
python bot.py
```

## Как работает
- Команда `/start` выводит кнопки «Проверить подписку» и «Повысить шанс».
- «Проверить подписку» дергает `getChatMember` и убеждается, что пользователь состоит в `@torgovlya_kfu`.
- «Повысить шанс» просит прислать скрин истории/репоста; бот фиксирует получение фото и отвечает подтверждением.

Настройки целевого чата меняются в константе `TARGET_CHAT` внутри `bot.py`.
